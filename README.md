Este proyecto es una aplicación en Python que permite gestionar una lista de clientes mediante un menú interactivo en consola.



**El sistema permite:**



Agregar clientes (nombre y cédula)
Eliminar clientes
Listar todos los clientes registrados

Buscar un cliente por nombre

Determinar el país de origen según las dos primeras letras de la cédula

El programa utiliza estructuras de datos como diccionarios para almacenar la información de forma dinámica.



**Requisitos**



1\. Tener Python instalado (versión 3.13.7 recomendada o superior)



* **Clonar o descargar el repositorio:**

	git clone <URL\_DEL\_REPOSITORIO>

* **Acceder a la carpeta del proyecto:**

	cd nombre\_del\_proyecto

* **Ejecutar el archivo principal:**

1. &nbsp;	python menu\_python.py
2. &nbsp;	O en algunos sistemas: python3 menu\_python.py
